## jrfront_recruitment-task

### Prerequisites

* Node js
```sh
https://nodejs.org/dist/v12.15.0/node-v12.15.0-x64.msi
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/krzosdominik/jrfront_recruitment-task.git
```
2. Install NPM packages
```sh
npm install
```
3. Run gulp
```sh
gulp
```
